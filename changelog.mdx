## Release History

RevTrack evolves rapidly to deliver precise commission automation. You find detailed release notes below, organized by version. Each update includes new features that enhance your sales data integration, improvements for better performance, and bug fixes to ensure reliability.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Real-time commission previews: View calculated commissions as sales data updates, without full recalculations.
- Custom commission tiers: Define up to 5 tiers per salesperson with percentage-based rules.

## Improvements

- Faster data processing: Reduced calculation time by 40% for datasets over `1000` sales records.
- Enhanced dashboard: Added export options for CSV and PDF commission reports.

## Bug Fixes

- Resolved sync issues with Salesforce integrations during peak hours.
- Fixed rounding errors in multi-currency commission calculations.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- API webhook support: Receive instant notifications on commission changes via your endpoint.
- Role-based access: Assign view-only or admin permissions to team members.

## Improvements

- UI optimizations: Improved mobile responsiveness for commission tracking on the go.
- Data validation: Automatic checks for duplicate sales entries before processing.

## Bug Fixes

- Corrected filter persistence when switching between salesperson views.
- Patched security vulnerability in session handling (reported via our bug bounty program).

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core commission engine: Automate calculations from CSV uploads or CRM integrations like HubSpot and Salesforce.
- Salesperson dashboard: Track individual and team commissions with historical trends.

## Breaking Changes

- Updated config format: Migrate from XML to JSON for commission rules. See migration guide below.

## Initial Setup

Run this to update your rules file:

<CodeGroup tabs="JSON,XML (Legacy)">
```json
{
  "rules": [
    {
      "tier": 1,
      "min_sales": 0,
      "max_sales": 10000,
      "rate": 0.05
    }
  ]
}
```
```xml
<!-- Legacy format - migrate to JSON -->
<rules>
  <tier min="0" max="10000" rate="5%"/>
</rules>
```
</CodeGroup>

</Update>
